# ずんだもん作業タイマー (Zundamon Task Timer) 👋

VOICEVOXを利用して、ずんだもんが作業スケジュールを音声で読み上げてくれる、実務・夜勤対応型のタスクタイマーです。

## 🚀 概要
「ただのタイマーでは味気ない」
そんな思いから、VOICEVOXのAPIと連携し、設定した時間になるとずんだもんが優しく（？）タスクを教えてくれる仕組みを構築しました。
夜勤などの日付を跨ぐスケジュール管理にも対応しています。

## ✨ 特徴
* **VOICEVOX連携**: API経由でリアルタイムに音声を生成。設定したセリフを喋ります。
* **進捗の可視化**: 経過時間に合わせて進捗バーがリアルタイムに上昇します。
* **夜勤・24時間対応**: 日付跨ぎのロジックを搭載。深夜帯の業務管理もスムーズです。
* **簡単モード切り替え**: フォルダ内のCSVを読み込み、複数の作業パターンを即座に切り替え可能。

## 🛠 使い方
1. **VOICEVOXを起動**: 本ツールを実行する前に、VOICEVOXを立ち上げておいてください。
2. **CSVで設定**: 
   - `時間,タスク名,セリフ` の形式でCSVを作成します（例：`21:00,巡回,巡回の時間なのだ！`）。
3. **実行**: モードを選択してタイマーを開始するのだ！

## 🛠 技術スタック
- **Language**: Python 3.x
- **GUI**: Tkinter
- **Audio**: Pygame (mixer)
- **Voice Engine**: VOICEVOX API
