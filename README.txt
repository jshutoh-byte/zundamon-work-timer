===============================================================
ずんだもん作業タイマー (Zundamon Task Timer)
===============================================================

作成者：J. Shuto (@jshutoh-byte)
バージョン：1.0
日付：2026-01-14

---------------------------------------------------------------
■ 概要
---------------------------------------------------------------
本ツールは、VOICEVOXを利用して設定時刻に音声を再生する、
実務特化型のタスクタイマーです。

夜間業務やルーチンワークのスケジュールをCSVで管理し、
ずんだもんの音声でナビゲートします。

---------------------------------------------------------------
■ 準備・動作環境
---------------------------------------------------------------
1. VOICEVOX（無料）をインストールし、起動しておいてください。
2. Python環境で以下のライブラリが必要です。
   - requests / pygame
3. 同階層にあるCSVファイルを自動で読み込みます。

---------------------------------------------------------------
■ 操作方法
---------------------------------------------------------------
・起動後、リストから実行したいモード（CSV名）を選択。
・「ずんだもんの声 ON」のチェックで、音声の有無を切り替え可能。
・Enterキー等の入力は不要。時間になれば自動で喋ります。

---------------------------------------------------------------
■ CSVの書き方
---------------------------------------------------------------
1列目：時間（HH:MM）
2列目：タスク名
3列目：喋らせたいセリフ（空欄の場合はタスク名を読み上げます）

例：
23:30,データバックアップ,データのバックアップを忘れないでほしいのだ。
